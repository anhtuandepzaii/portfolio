# 🚀 Personal Portfolio - Dao Anh Tuan

Welcome to my personal portfolio repository! This project is a professional showcase of my skills, projects, and journey as a **Software Engineering** student at Quy Nhon University (QNU).

## 🎨 Design Overview
This portfolio is built with a "Developer-First" aesthetic, combining clean lines with high-performance design principles.
- **Theme:** Minimalist Dark Mode.
- **Color Palette:** - `Primary Red (#d32f2f)` - For branding and accents.
  - `Deep Background (#191d17)` - For a comfortable dark experience.
  - `Text Ivory (#e1e4da)` - For high-contrast readability.
- **Typography:** - **JetBrains Mono:** Used for headers and technical elements to reflect a coding environment.
  - **Roboto:** Used for body text to ensure modern clarity.

## 🛠️ Tech Stack
- **Languages:** HTML5, CSS3 (Flexbox & Grid), JavaScript (ES6+).
- **Architecture:** Responsive Web Design (RWD) with a modular CSS structure.
- **Tools:** VS Code, Git/GitHub, and specialized system tools.
